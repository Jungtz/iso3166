# iso3166
Codes for the representation of names of countries and their subdivisions
* iso3166-1 Country codes, ex: TW
* iso3166-2 City codes, ex: TW-TPE
* ~~iso3166-3~~

## Project setup
```
npm install
```

### Compiles
```
node app.js
```
